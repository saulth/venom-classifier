Sobre el clasificador

El desarrollo de Machine Learning es utilizado para la identificación de patrones dentro de una serie de datos. En este caso se planea ejecutar un modelo de Machine Learning que a partir de los datos proporcionados dentro de las bases de datos con información de venenos, desarrollar una aplicación que permita una fácil interacción entre científico-máquina para la identificación de venenos a partir de una cadena de caracteres, que serán una cadena de proteínas o transcriptomas.

En el campo de la biología molecular, existe una rama llamada venómica, que son los estudios de proteómica de venenos. Esta rama se ha revelado como una estrategia adecuada para identiﬁcar y cuantiﬁcar las familias de toxinas presentes en los venenos.
